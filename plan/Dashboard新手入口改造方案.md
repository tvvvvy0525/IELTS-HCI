# Dashboard 新手入口改造方案 (已更新)

## 1. 目标
优化 Dashboard 的新手入口（雅思入门）的展示逻辑和位置，提升新用户的引导体验，同时允许熟练用户隐藏该入口。采用纯 CSS 极简操作演示动画在弹窗中展示如何再次找到该入口，提升界面的高级感。

## 2. 拟修改文件
- `src/views/DashboardView.vue`
- `src/views/ExamLayout.vue`

## 3. 详细设计

### 3.1 DashboardView.vue 修改
- **位置调整**：将 `intro-entry-card` 移动到 `<div class="dashboard-view">` 的最顶部。
- **UI 改造**：
  - 按钮一：“进入雅思入门”。
  - 按钮二：“不再显示”。
- **交互逻辑**：
  - 点击“不再显示”后：
    1. 弹出一个自定义的精美弹窗。
    2. 弹窗内包含文字：“已为您隐藏该入口。如需再次访问，可从侧边栏-雅思入口进入。”
    3. **弹窗内嵌纯 CSS 极简动画**：模拟一个极简的侧边栏。动画循环播放：一个小手光标（Cursor）从上方移动到侧边栏的“雅思入门”项上，点击后该项产生高亮闪烁效果。
    4. 用户关闭弹窗或点击确认后，新手入口卡片隐藏，并持久化至 `localStorage`。

### 3.2 ExamLayout.vue 修改
- **侧边栏联动**（可选，作为弹窗动画的补充）：
  - 在用户点击“不再显示”的瞬间，除了弹窗，侧边栏真实的“雅思入门”项也可以进行一次短暂的闪烁，作为真实 UI 的指引。

## 4. 验证计划
- 刷新页面，确认“新手入口”卡片出现在 Dashboard 最顶部。
- 点击“进入雅思入门”，确认能正确跳转。
- 点击“不再显示”，确认弹出精美弹窗，且弹窗内有纯 CSS 的操作演示动画。
- 确认再次刷新页面后，卡片不再显示。
